Site Info

This website is built with Eleventy, a static site generator using VSCode. It is hosted on Neocities and uses deploy-to-neocities github action.

Snails is mobile-friendly, responsive, and best viewed on firefox browsers.

Credits

All font and image credits can be found in my styles.css file, which you can also view on github.

Resources

Things that helped me build this website. Full list of resources and bookmarks can be found on my Links page.

General Web Development

  • Bootstrap Icons – Large set of free, open-source icons.
  • Luxon – Date/time library for JavaScript.
  • Normalize.css – CSS resets for consistent styling across browsers.
  • html-minifier-terser – Tool to minify and optimize your final HTML files.
  • Eleventy – Easiest static site generator to learn.
  • Visual Code Studio – Great offline code editor with lots of plugins/customization.

Tutorials

  • Petrapixel's Eleventy Tutorial – Neocities-friendly guide for getting started with Eleventy.

Image Editing

  • Dither it! – Online tool for dithering images.
  • Dither Me This – More advanced image dithering and pixel manipulation tool.

Eleventy Plugins

  • external-links – Automatically adds target and rel attributes to external links in Eleventy.
  • markdown-it-anchor – Adds anchors to your Markdown headers for easy linking.
  • markdown-it-attrs – Lets you add custom classes and attributes directly in Markdown.
  • eleventy-plugin-toc – Generate a table of contents with easy config and customization.
  • eleventy-fetch – Fetch remote content and cache it locally, useful for pulling in data from APIs or external sources at build time.
  • rss – Adds filters and shortcodes for generating RSS feeds.
  • reading-time – Estimates and displays reading time for blog posts, based on word count and average reading speed.
